## 📝 Project Overview
This project investigates customer attrition for **Databel**, focusing on identifying patterns, reasons for churn, and high-risk segments. Using **Power BI**, I processed data for **6,687 customers** to provide actionable business intelligence aimed at increasing retention rates.

## 🚀 Key Business Insights
* **Churn Rate:** Analyzed a baseline churn rate of **26.86%**.
* **The Competitor Factor:** **44.82%** of churned customers were lost to competitors, indicating a need for market-driven strategy adjustments.
* **Contract Vulnerability:** **Month-to-Month** contracts account for **51.01%** of the base and show the highest probability of exit.
* **Service Friction:** Identified a direct link between customer support interaction and churn, with an average of **1.36 service calls** per customer.

## 🛠️ Technical Toolkit
* **Data Transformation:** Leveraged **Power Query** for data cleaning and ETL processes.
* **Data Modeling:** Established a robust data model to support complex cross-filtering.
* **Advanced DAX:** Created measures for:
    * Dynamic Churn Percentage
    * Customer Segmentation
    * Service Call Correlation
* **Interactive Design:** Implemented slicers, tooltips, and drill-throughs for a user-centric experience.
